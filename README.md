A premium, high-performance personal portfolio built with a focus on aesthetics, smooth animations, and modern web technologies. This project showcases my work as an **AI Developer** specializing in Multimodal AI, RAG systems, and Scalable Backends.

## ✨ Features

- **🎯 Interactive Project Showcase**: Detailed breakdowns of complex AI projects with impact metrics.
- **🎭 Smooth Animations**: Powered by Framer Motion for a fluid user experience.
- **🌓 Modern UI/UX**: Built with Tailwind CSS v4 and shadcn/ui, featuring a sleek dark-mode aesthetic.
- **📱 Fully Responsive**: Optimized for all screen sizes, from mobile to ultra-wide displays.
- **⚡ Performance First**: Optimized images, code splitting, and fast page loads via Next.js 15.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/) & [Base UI](https://base-ui.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Package Manager**: [pnpm](https://pnpm.io/)

## 🚀 Getting Started

### Prerequisites

- Node.js 20.x or higher
- pnpm (recommended) or npm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kaisarfardin6620/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   pnpm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file based on `.env.example`.

4. **Run the development server:**
   ```bash
   pnpm dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
